🎓 Student Result Management Dashboard (SRMS)

A responsive, single-page dashboard application for managing students, courses, and results.
Built with HTML, TailwindCSS, and Chart.js, this project demonstrates interactive data visualization and CRUD-style operations with a clean UI.

🚀 Features

Dashboard Overview

Total Students, Courses, Average Score, Passing Rate

Grade distribution visualized with a doughnut chart

Student Management

Search, sort, and manage student records

View individual student details

Visualize performance across courses with a bar chart

Add new students via modal forms

Course Management

List and manage available courses

Add new courses via modal forms

Interactive UI

Smooth navigation between Dashboard, Students, and Courses

Clean TailwindCSS-based styling

Responsive layout for mobile and desktop

🛠️ Tech Stack

Frontend: HTML5, TailwindCSS

Charts: Chart.js

Fonts: Google Fonts (Inter)

JavaScript: Vanilla JS for interactivity and state management

📂 Project Structure
.
├── index.html    # Main HTML file containing the entire app
├── README.md     # Project documentation

⚙️ How It Works

Uses mock data for students, courses, and results (can be extended to APIs later).

JavaScript handles:

State management

Rendering charts dynamically

Handling modals for data entry

Sorting, searching, and filtering

All visualizations update in real-time as data changes.

📊 Visualizations

Dashboard Cards: Quick stats (students, courses, average score, passing rate)

Grade Distribution Chart: Donut chart comparing grade proportions

Student Performance Chart: Bar chart for individual student scores
